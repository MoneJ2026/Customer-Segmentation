# Customer Segmentation using K-Means

## Project Description

This project uses the K-Means clustering algorithm to segment customers based on their Income and Spending Score.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Project Structure

```
Customer-Segmentation/
│── data/
│── clustering.py
│── requirements.txt
│── README.md
│── .gitignore
```

## How to Run

```bash
pip install -r requirements.txt
python clustering.py
```
